# LangQuest Product Website

A modern, responsive product landing page developed to showcase **LangQuest**, a Flutter-based mobile application created for the **Mobile Application Programming** course.

The website introduces the application's purpose, highlights its core features, presents the development team, and provides contact information in a clean and interactive interface.

---

## 📖 About LangQuest

**LangQuest** is a game-based learning and event management platform designed for university students, with an initial focus on supporting the **UTM English Club**.

The application aims to solve common challenges faced by student organizations, including:

* Low student engagement
* Scattered announcements across messaging platforms
* Manual attendance tracking
* Limited insight into member participation
* Lack of motivation in language learning

By combining event management, gamified English learning, and progress tracking, LangQuest provides a centralized platform for both students and administrators.
